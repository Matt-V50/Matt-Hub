Subject API
用法说明
tio.Subject(image=tio.ScalarImage(...), label=tio.LabelMap(...))image 和 label 绑定，transform 同步应用
subject = transform(subject)返回新 subject，原始不变
subject['image'].datashape: (C, D, H, W)
subject['image'].affine4×4 affine matrix (voxel → world)
常用 Transforms
Transform关键参数备注
tio.ToCanonical() — 重排 axis → RAS+，⚠ axis ordering 会变
tio.Resample(target) target: spacing tuple 或 reference image 传 reference image 是最稳的 inverse 方式
tio.CropOrPad(size) size: (D,H,W) 保持 affine 正确更新
tio.ZNormalization() masking_method 只对 ScalarImage 生效
tio.RescaleIntensity() out_min_max=(0,1) —
