SAM-Med3D: Towards General-Purpose Segmentation Models for Volumetric Medical Images 分割 2024-03-12

将 SAM 扩展到 3D 医学图像分割。核心改动是把 2D image encoder 替换成 3D ViT，同时引入 3D prompt encoder 支持点、框提示。

训练数据用了 21k+ 个 3D 医学图像样本，涵盖 CT/MRI 多个模态。zero-shot 能力在通用分割任务上显著优于之前方法。

注意事项：输入需要 resample 到 1mm isotropic，坐标系对齐是关键，ToCanonical 之后 axis ordering 会变。

TorchIO: a Python library for efficient loading, preprocessing, augmentation and patch-based sampling of medical images in deep learning 工具 2024-01-05

Subject-based API 是关键设计——image 和 label 共享同一个 transform，保证空间对齐。不要对 image/label 分别跑 transform。

Resample(reference) 是最可靠的 inverse transform 方式，比手动算 affine 矩阵靠谱得多。
