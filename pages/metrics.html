IoU (Intersection over Union) 和 Dice 系数都是衡量两个区域重叠程度的指标，广泛用于目标检测、语义分割、实例分割任务。

几何交互 · 拖拽体验 IoU & Dice
橙色 = Ground Truth，蓝色 = Prediction。拖动蓝色框调整位置和大小。
IoU vs Dice 的关系
Dice = 2·IoU / (1 + IoU)
IoU = Dice / (2 − Dice)
F1 Score = Dice 系数（在二值分割中等价）
IoU 阈值判定
TP = 交集面积
FP = 预测区域 − 交集 (过预测)
FN = GT区域 − 交集 (漏检)
IoU = TP / (TP+FP+FN)
IoU = |A ∩ B| / |A ∪ B|  =  TP / (TP+FP+FN)     Dice = 2|A ∩ B| / (|A|+|B|)  =  2TP / (2TP+FP+FN)
IoU ↔ Dice 转换曲线
Dice 始终 ≥ IoU（相同重叠时 Dice 分值更高）。
当 IoU→0，Dice→0；当 IoU→1，Dice→1；
IoU=0.5 时 Dice≈0.667。
像素级分解示意
绿色 TP：预测和GT都为正
橙色 FP：预测为正，GT为负（过预测）
橙色 FN：预测为负，GT为正（漏检）
白色 TN：双方都为负
